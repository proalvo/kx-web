# kx-web
This is public results website for Kayak Cross competitions (companion to KX-Results)

This is public repository for the **kx-web** software, but we have good news - **you do not need to install this software, if you use [wwcf.fi/kx-results](https://wwcf.fi/kx-results) for your competitions then you youc publish the results in real time**.

Before you can publish the the results, you need to do following:
1. install the kx-results
2. send an email to kari.nykanen@wwcf.fi to get organisation key for the kx-results. This enables to publish results on *wwcfi.fi/kx-results*.
3. organize you competition.
